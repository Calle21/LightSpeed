Indentation is important in Nova. Some restrictions are also made on the code simply to make it look better. Here they are.

Assignments

                a   = 4
                abc = 5
                ab  = 6

When making assignments, the equal signs have to be matched up like this.

If statements

                a < abc   >> ...
                abc == ab >> ...
                ab <= a   >> ...
                ...

The >> has to be aligned like this
