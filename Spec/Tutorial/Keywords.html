There are some keywords in Nova that are treated uniquely by the compiler. There keywords may not be used as identifiers. There are currently no ways to create your own keywords in Nova

The keywords are...

    ?
    type

?

This is the if operator. It has been explained in the introduction

type

This keyword is written at the toplevel in order to create types. Other keywords exist to create types, however, type is the general keyword with which all types can be defined.
