Indentation

Indentation plays a big part in Nova. To demonstrate, here is an example program.

                Int biggestofthree Int a Int b Int c
                  biggestoftwo (biggestoftwo a b) c
                  where
                  Int biggestoftwo Int a Int b = a > b ? a
                                                 b

The body of a function begins either after an equals sign (with precicesly one space in between), or exactly two spaces in on the next line.
