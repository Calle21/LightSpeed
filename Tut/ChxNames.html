Chapter 3 - Names

These are the types of names, or symbols, that exist in Nova

    Function names

    Function names consist of a string of lower case letters and numbers, where at least one character is a letter. A program to test whether a string is a legal function name could look like this

    Bool testname String s
  all (lower || number) s
  where
  Bool lower Char c = c >= 'a & c <= 'z
  Bool number Char c = c >= '0 & c <= '9

    Type variables
    Type variables consist of only upper case letters.

    Infix function names Functions can be infix operators. These names consist of either the special symbols + - * or a normal function name surrounded by back-quotes `infix`
